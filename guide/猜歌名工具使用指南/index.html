快速导览

的文本，点击 Import 按钮导入歌曲列表。

  1. 点击 Sort By Length 按钮。
  2. 点击 Copy to Clipboard 按钮，将输出复制到剪贴板。
  3. 开字符：在 guess 框里输入被开的字符，然后按回车。
  4. 猜出歌：选中歌名左边的框，表示这首歌被猜出。
  5. 点击歌名最右边的框，隐藏被猜出的歌（不一定要这么干）。

完整介绍

Songs List:

在这里管理题目（歌名），点击下面的 + 添加一条。

添加条目后，每个条目从左到右分别是：

  1. 单选框，表示这首歌是否已经被猜出来了。底下的框表示全选 / 反选。
  2. 歌名
  3. x 按钮，删除这首歌
  4. hide 单选框，一首歌被 hide 后不会出现在 text 里，但是编号的时候不会跳过他（用于隐藏已经被猜出来的歌）。

加入歌曲后按 Sort by length 即把所有歌曲按照歌名的长度排序。

Output

输出框，点击右边的按钮复制到剪贴板

guess

在框里输入被开的字符，加不加空格都可以

输入后按回车即会刷新，也可以按 refresh 按钮刷新

右边的 auto sort 单选框表示对于被开的字符，按照字典序自动排序。
